##### 1.Computed 和 Watch 区别

| 区别 | watch | computed |
| :--: | :--:   | :--: |
| 用法上 | 根据响应式数据作出相应的变化 | 依赖响应式数据的变化而变化，生成了一个新的响应式数据 |
| 特点上 | 无缓存 | `有缓存`，无副作用 |
| 场景上 | 可`异步`，消耗较大 | 用于频繁变动的数据 |
| 原理上 | 相当于属性 `setter` 的回调函数 | 相当于设置了属性的 `getter` |
