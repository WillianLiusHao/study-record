# 广州 6 年前端 · 1 个月面试冲刺计划

以下计划默认你每周投入 6 天、每天 6～7 小时，可根据个人节奏微调。
总原则：

1. 先广度再深度，先体系后刷题；
2. 每天「早-中-晚」三段：上午理论，下午编码，晚上复盘+算法；
3. 每周末做 1 次全真模拟面试（技术+HR），产出 1 篇可对外分享的技术总结，既复盘又当博客素材；
4. 所有代码 push 到 GitHub 私有仓，方便随时展示给面试官。



## 前置准备（第 0 天，今晚就完成）
注册 1 个新 GitHub 私有仓 `interview-2025`，建 3 个目录：

- algorithm

- system-design

- demo-code

- 准备 3 份简历（PDF+Word）：

  - 通用版（3 年经验外观，突出广度）

  - 资深版（6 年深度，突出性能优化、架构）

  - 管理版（如有带团队经历，突出跨端协调）

- 收集目标 JD：拉勾/BOSS 上收藏 30 条「广州 18k–30k 前端」JD，用 Notion 建表，拆解关键词（Vue3/React18、Vite、微前端、低代码、Node 全栈、性能指标、CI/CD）。后面复习围绕这些关键词展开。



## 第 1 周 「基础加固 + 算法热身」

目标：扫清高频八股盲区，算法从 0 到 60 分，建立每日节奏。

| 时间段           | 任务清单                                                     | 产出                  |
| ---------------- | ------------------------------------------------------------ | --------------------- |
| 上午 09:30-12:00 | ① ES2023 新特性速览（Array.fromAsync、toSorted、findLast）<br>② 闭包/作用域/事件循环 3 篇优质博文精读 + 手写题 5 道 | 笔记.md + 手写题 push |
| 下午 14:00-17:00 | ③ Vue3 & React18 源码切片（只啃响应式 & 调度器，不求多，只求透）<br>④ 性能：重排重绘、防抖节流、图片懒加载手写 | CodePen 各 1 个 demo  |
| 晚上 20:00-22:00 | ⑤ LeetCode 经典 12 题（数组+链表+哈希）<br>⑥ 复盘 30 min，写「今日难点速记」 | 算法仓 commit         |

周末（第 6 天）

- 上午：Mock Interview 1 —— 自我介绍 + 项目深挖（STAR 模板录屏）
- 下午：把本周所有手写题整理成《前端基础 30 题》博客，发布掘金/知乎，引流简历。



## 第 2 周 「工程化 + 性能优化」

目标：让简历出现「工程化」「性能指标」「CI/CD」关键词，能讲出落地数据。

| 时间段 | 任务清单                                                     | 可量化产出              |
| ------ | ------------------------------------------------------------ | ----------------------- |
| 上午   | ① Vite 插件开发实战：写 1 个自动压缩图片的插件，输出本地 npm 包<br>② Webpack 与 Vite 差异对比表（启动/热更新/产物/插件机制） | 插件 github + 差异表.md |
| 下午   | ③ 性能指标采集：LCP/FID/CLS 代码注入 + 上报 Node 服务（Express+Mongo）<br>④ Chrome DevTools Performance 面板 5 个 Case 重演 | 指标看板截图            |
| 晚上   | ⑤ 算法：栈/队列/二叉树 12 题<br>⑥ 复盘：把性能优化写成 1 页 A4 脑图 | 脑图.png                |

周末

- Mock Interview 2 —— 系统追问：
  “首屏 3G 网络下如何做到 1.5s 内完全可交互？”
  必须给出数字：骨架屏占比、JS 分包数量、图片 WebP 体积、HTTP2 多路复用。
- 博客 2：「从 0 搭建前端性能监控平台」发掘金，附 GitHub 源码。



## 第 3 周 「Node 全栈 + 微前端 + TypeScript 深度」

目标：补全「能独立上线项目」能力，回答「你如何部署」「如何回滚」。

| 时间段 | 任务清单                                                     | 产出              |
| ------ | ------------------------------------------------------------ | ----------------- |
| 上午   | ① Node 高并发：cluster + pm2 + nginx 负载均衡本地复现<br>② 写 1 个 JWT 登录体系（含 refreshToken、黑名单、Redis） | 接口文档 Swagger  |
| 下午   | ③ 微前端：qiankun 实战，把上周的 Vue3 子应用和 React18 子应用整合到基座<br>④ 沙箱隔离原理 & 样式隔离手写实现 | 微前端仓库        |
| 晚上   | ⑤ TypeScript 类型体操 15 题（Pick、Trim、DeepReadonly、模板字面量）<br>⑥ 算法：动态规划入门 8 题 | ts-challenge 提交 |

周末

- Mock Interview 3 —— 代码 Review：
  让朋友提 PR，你 15 分钟给出 5 条 comment（类型、性能、可读性、边界、测试）。
- 博客 3：「qiankun 样式隔离源码级解析」发知乎。



## 第 4 周 「面试冲刺 + 高频算法 + 薪酬谈判」

目标：每日至少 2 场真实面试（视频/现场），把算法正确率刷到 80%，HR 面稳过。

| 时间段 | 任务清单                                                     | 备注                 |
| ------ | ------------------------------------------------------------ | -------------------- |
| 上午   | ① 算法：LeetCode Top 100 剩余 30 道 + 字节/阿里/腾讯近年真题<br>② 手写：Promise.all、深拷贝、EventEmitter、ES5 继承 | 每题计时 15 min      |
| 下午   | ③ 投递+面试：每天 2 场，面完 30 min 内复盘，把不会的问题写入「错题本」<br>④ 系统设计：统一复习「短链系统」「聊天室」「低代码平台」3 套方案 | 用 draw\.io 画架构图 |
| 晚上   | ⑤ HR 面模拟：自我介绍 1 min/3 min/5 min 三版本；离职原因、职业规划、薪酬报价话术<br>⑥ 错题本回炉，对应知识点再录 3 分钟短视频放 Notion | 形成可检索知识库     |



## 每日时间模板（可微调）

09:30-12:00  理论+源码
14:00-17:00  编码+项目
20:00-22:30  算法+复盘

## 工具推荐

- 刷题：LeetCode-cn + 力扣打卡插件（vscode-leetcode）
- 算法视频回放：OBS 录屏，回看自己讲思路
- 简历/博客：Notion → 一键导出 PDF & 静态博客
- 架构图：draw.io 插件直接存 GitHub
- 面试排期：用飞书日历，提前锁定 HR 时间，防止撞车

## 预期结果

1. GitHub 私有仓 ≥ 8 个有意义 commit，可供现场演示；
2. 掘金/知乎技术文章 ≥ 3 篇，总阅读量 5k+（广州圈子很小，面试官常搜名字）；
3. LeetCode 刷题量 ≥ 150，周赛排名进前 30%；
4. 真实面试 ≥ 8 场，拿到 ≥ 2 个 offer，薪酬区间 22k–28k（对标广州 6 年）。
