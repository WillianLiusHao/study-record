    以下是为一位 6 年经验、坐标广州、刚离职、无需复习算法 的前端工程师，量身定制的 1 个月冲刺复习计划。 目标：1 个月内把“高频面试考点 + 项目亮点 + 简历亮点”全部梳理清楚，具备随时面试、随时能讲清楚、能写清楚、能调通代码的状态。

    # 🎯 总原则

    • 不复习算法（如你所愿）
    • 面向面试：只复习“广州市场 90% 公司会问”的内容
    • 每日 6~7 小时，一周 6 天，留 1 天机动/休息
    • 输出驱动：每天必须落地下列至少 1 份可交付物
      • 代码片段（GitHub 私有库）
      • 思维导图/笔记（语雀/Notion）
      • 录音/录像自我介绍或项目讲解（手机即可）
    • 简历并行：第 1 周完成 V1.0，第 4 周完成 V4.0 定稿

    # 📅 四周时间轴

    周次 主题 目标 关键产出
    Week1 基础扫盲+简历骨架 把“所有会被问的基础”快速过一遍，搭好简历框架 简历 V1、基础脑图、10 个高频代码片段
    Week2 框架深度+项目包装 React 源码级、Webpack 优化、项目难点梳理 简历 V2、项目 STAR 表、3 段 3 分钟项目视频
    Week3 工程化+性能+场景题 性能调优、微前端、CI/CD、监控、场景题 简历 V3、性能 Case 文档、场景题 30 题录音
    Week4 全真模拟+查漏补缺 模拟面试 5 场，把不会的全部补回来 简历 V4、问题清单清零、 Offer-ready 状态

    # 📌 每日节奏（6~7h）

    时间段 时长 任务 工具/备注
    09:00-09:30 0.5h 昨日复盘 Notion 打勾/划掉
    09:30-12:00 2.5h 主题输入+代码验证 Chrome DevTools + GitHub
    12:00-13:30 午休
    13:30-15:30 2h 输出：笔记/录音/简历片段 语雀、飞书妙记
    15:30-16:00 0.5h 运动/散步 保证状态
    16:00-18:00 2h 刷面经+口述答题 小程序“前端面试江湖”、Boss 面经
    晚上 自由 可机动/休息/看剧

    # Week1 详细计划（Day1-Day7）

    Day 模块 必过清单 代码/输出要求
    1 JS 基础 事件循环、闭包、原型链、Promise、async/await 手写 Promise.all/race、事件循环画图
    2 CSS/浏览器 BFC、重排重绘、Flex/Grid、层叠上下文、移动端 1px 写 3 个可视化 demo 录屏
    3 DOM/网络 跨域、缓存、HTTPS、三次握手、从输入 URL 到渲染 画一张“从 URL 到像素”思维导图
    4 React 基础 生命周期、Hooks 原理、Fiber 架构、setState 同步还是异步 手写 useReducer+useContext 迷你 Redux
    5 Vue 基础 响应式原理、computed/watch、nextTick、diff 策略 手写迷你响应式（Proxy）
    6 简历+GitHub 完成简历 V1.0 + 把上述 5 天代码 push 到私有库 README 写“项目运行一行命令”
    7 休息/机动 查漏补缺/运动/家庭日 把本周最难的 5 题录音讲解

    # Week2 详细计划（Day8-Day14）

    Day 模块 必过清单 代码/输出要求
    8 React 进阶 虚拟 DOM 实现、diff 算法、Concurrent Features、React18 新 API 手写虚拟 DOM→真实 DOM 100 行
    9 状态管理 Redux 中间件、Zustand、Jotai、Recoil、React-Query 用 React-Query 写“分页+缓存+乐观更新”demo
    10 路由+权限 React-Router v6 源码、权限方案、面包屑、Tabs Keep-alive 写一套“按钮级权限”hooks
    11 项目难点梳理 从过去 3 个项目里挖出 5 个“有量化结果”的难点 输出 STAR 表格（Situation-Task-Action-Result）
    12 Webpack/Vite 热更新原理、分包、压缩、Tree-Shaking、插件编写 写个“自动生成 CDN 外链”的 Vite 插件
    13 微前端 qiankun 沙箱原理、样式隔离、主应用性能监控 搭个 qiankun 基座+两个子应用，输出 README
    14 休息/机动 把本周代码整理成“项目集锦”页面 录 3 段 3 分钟项目讲解视频（手机即可）

    # Week3 详细计划（Day15-Day21）

    Day 模块 必过清单 代码/输出要求
    15 性能优化 Lighthouse 打分、FCP/LCP/TBT、Code-Split、懒加载、虚拟滚动 把 Week2 的 demo 优化到 90 分，写 Case 文档
    16 监控体系 Sentry sourcemap、阿里 ARMS、埋点方案、性能日志 写个“错误+性能”一体化上报 SDK（100 行）
    17 CI/CD GitHub Actions + Docker + 阿里云 OSS 自动化部署 把 Week2 的微前端项目一键部署
    18 场景题 “首屏 3s 优化”“白屏检测”“H5 秒开”“SEO 方案” 整理 30 题场景题+口述答案录音
    19 小程序/Taro 微信小程序原理、Taro 编译流程、性能优化 用 Taro 3 写个“列表+虚拟滚动”demo
    20 跨端+基建 Electron、Flutter Web、Monorepo、Changeset 用 Turborepo 搭 Monorepo，管理 2 个子包
    21 休息/机动 把本周所有 Case 整理成“线上文档” 生成二维码，手机随时复习

    # Week4 详细计划（Day22-Day30）

    Day 模块 必过清单 代码/输出要求
    22 模拟面试1 找好友/群友，30 min 自我介绍+项目+追问 录音+问题清单
    23 模拟面试2 Boss 直聘/牛客网语音面，收集真实问题 把答不上来的写进“错题本”
    24 模拟面试3 现场手写代码（无算法）：事件总线、深拷贝、防抖节流 录屏+代码 push
    25 模拟面试4 压力面：质疑项目难点、数据真实性 准备“数据佐证”材料（截图、邮件、Git 记录）
    26 模拟面试5 HR 面：离职原因、职业规划、期望薪资 准备 3 档薪资方案（保底/理想/冲刺）
    27 查漏补缺 把 5 场面试所有错题全部清零 每题写 100 字“标准答案”+口述录音
    28 简历定稿 V4.0 定稿，找 3 位资深前端帮看 PDF+在线 Markdown 双版本
    29 装备检查 GitHub 私有库转公开、掘金/知乎发技术总结、领英更新 生成“个人作品导航页”
    30 休息/庆祝 剪个 1 分钟 vlog 纪念“离职冲刺月” 晚上投出第一批 10 份简历

    # 📣 温馨提示

    1. 广州市场特点
      • 小程序、混合 App、B 端后台、电商业务偏多，Week3 加入小程序&跨端是加分项。
      • 薪资区间：6 年普遍 18k-28k（优秀可冲 30k+），提前准备好 3 档报价。
    2. 简历控制 2 页，项目亮点用“数据”说话：
      • “首屏从 4.2s 优化到 1.8s，Lighthouse 打分 +32”
      • “权限系统覆盖 2000+ 菜单、100+ 数据权限，零线上事故”
    3. 离职空窗解释：“主动离职，1 个月系统复盘+技术升级，现已 Ready”——HR 最怕你“被动离职+能力焦虑”。
    4. Day30 当天就开始投简历，不要等“完美”，广州招聘节奏快，早投早面早拿 Offer。

    祝你 30 天后拿到“谈资满满、代码可验、面试对答如流”的状态，广州 Offer 多多！
