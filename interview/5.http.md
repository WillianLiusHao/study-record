## Http
### 1. 浏览器输入 url 到页面展示的过程？
[参考文章](https://juejin.cn/post/6844904054074654728#comment)
  1. URL 解析
  2. DNS 查询（DNS 域名查询系统转换成 IP 地址）
      - dns 服务器特点：高可用、高并发、分布式
      - 客户端 & 本地服务器 `递归查询`
      - DNS 服务器之间 `迭代查询`  

  3. TCP 链接
      - 通过`三次握手`，建立 TCP 连接

  4. 处理请求
      - 浏览器发起 HTTP 请求，目标服务器接收到 HTTP 请求并处理

  5. 接收响应
      - 目标服务器往浏览器发回 HTTP 响应

  6. 浏览器解析并渲染页面
      - ![](https://images.vrm.cn/ox/2022/06/08/浏览器渲染.png)
